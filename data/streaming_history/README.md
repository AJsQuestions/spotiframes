# Spotify Streaming History

Place your Spotify streaming history files here.

## How to get your data:
1. Go to https://www.spotify.com/account/privacy/
2. Scroll to "Download your data"
3. Request "Extended streaming history" (recommended) or "Account data"
4. Wait for email (can take up to 30 days for extended history)
5. Download and extract the ZIP file
6. Copy the JSON files here

## Expected files:
- **Extended format:** `endsong_0.json`, `endsong_1.json`, etc.
- **Basic format:** `StreamingHistory0.json`, `StreamingHistory1.json`, etc.

The extended format contains more data including track URIs.
